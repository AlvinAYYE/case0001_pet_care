## 專案目標

建立一個展示型網站，參考同類型寵物旅館網站的版面風格，提供品牌展示、服務介紹、圖片/海報展示與聯絡導流，並讓業主可自行於後台修改內容。
參考網站:[https://www.goodtimeph.com/](https://www.goodtimeph.com/)與[https://www.caliwoof.com/%E7%8B%97%E7%8B%97%E5%AE%89%E8%A6%AA%E4%BD%8F%E5%AE%BF](https://www.caliwoof.com/%E7%8B%97%E7%8B%97%E5%AE%89%E8%A6%AA%E4%BD%8F%E5%AE%BF)
## 專案範圍

### 前台

* 品牌主視覺
* 服務介紹
* 圖片 / 海報展示
* 店家資訊
* 聯絡 / 社群連結
* 支援手機與桌機瀏覽（RWD）

### 後台

* 可修改文字內容
* 可上傳 / 更換圖片與海報
* 可修改外部連結（LINE、IG、FB、地圖等）
* 可修改店家基本資訊（電話、地址、營業時間）

## 技術架構

* 前端：Vue 3 + TypeScript + Tailwind CSS + Vite + Axios
* 後端：PHP
* 資料庫：MySQL

## 不包含項目

* 金流
* 會員系統
* 完整預約系統
* Google 評論串接
* 複雜權限管理
* 多語系

## 驗收標準

* 前台可正常瀏覽，手機與桌機版面正常
* 後台可正常修改文字、圖片、連結
* 修改後內容可正確顯示於前台

## 備註

* 以參考網站的架構與風格為主，不直接複製原始素材
* 若後續新增功能或超出範圍需求，需另行評估工時與費用

品牌理念
恆寵愛 Perpetuity  
-長久陪伴、永恆的愛-



當毛孩年輕時，活潑、奔跑、充滿活力。
但當牠們慢慢變老，腳步變慢、需要更多休息與照顧時，
許多飼主會開始感到不安與焦慮。

我們理解那份心情。

恆寵愛 Perpetuity 的成立，
是希望提供一個更溫柔的選擇。

一個專門為高齡犬與需要安靜休養的狗狗
打造的照顧空間。

在一般寵物旅館中，
許多活潑的狗狗一起活動，
對高齡犬或個性害羞的毛孩來說，
往往容易造成壓力與疲累。

因此我們選擇提供 更安靜、更細心、更理解高齡毛孩需求的環境。

同時，我們也希望分享高齡犬照顧的經驗與知識，
陪伴飼主走過毛孩生命的每個階段。

------------------------------------

我們相信
陪伴毛孩不只是牠們年輕時的歡樂時光，
更是在牠們慢慢變老時
依然溫柔守護。

在這裡
每一份生命，都值得被好好照顧。
